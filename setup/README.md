# Setup

Choose which setup to run:

| Setup | For | Command |
|---|---|---|
| **Basic** | Live + batch classification | `setup_basic/setup_basic.bat` (.sh) |
| **Full** | ML model training | `setup_full/setup_full.bat` (.sh) |
| **Victim** | Target device for attacks | `setup_victim/setup_victim.bat` (.sh) |
| **Attacker** | Your machine to launch attacks | `setup_attacker/setup_attacker.bat` (.sh) |
